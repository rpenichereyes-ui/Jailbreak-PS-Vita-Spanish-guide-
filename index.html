HENlo - Un Jailbreak para PlayStation Vita

żQué es HENlo?

HENlo es el quinto jailbreak público para PlayStation Vita y PlayStation TV. Es similar a hacerle jailbreak a un iPhone o rootear un dispositivo Android. Te permite realizar modificaciones tanto de kernel como de usuario, cambiar la velocidad del reloj, instalar plugins, ejecutar homebrews y mucho más.


żQué le hará a mi dispositivo?

Cuando hagas clic en "Liberar", HENlo aprovechará un exploit en la consola y cargará el bootstrap henlo_jb, el cual podrás usar para iniciar el framework taihenkaku, así como instalar la herramienta VitaDeploy.
